Linux创建文件命令的方法

Posted on 2022-06-03 Edited on 2022-06-11

创建文件夹

  • mkdir 文件夹名
  • mkdir -p 文件夹名
  • md 文件夹名

创建文件

  • touch 文件名

查看当前文件夹下的所有目录

  • dir
  • ls

使用vim编辑器

  • vim 文件名(使用vim编辑器创建新文件之后会进入编辑器界面，然后输入你要的内容；
    最后按一下< shift + : >（在英文输入法模式下），输入wq回车即可保存退出。)
  • 在Linux系统中编辑文本文件，一定会用到VI/VIM编辑器，如何编辑时优雅的保存和退出?首先按ESC键跳到命令模式
  • :w - 保存文件，不退出vim。
  • :w file - 将修改另外保存到file中，不退出vim。
  • :w! - 强制保存，不退出vim 。
  • :wq - 保存文件，退出vim。
  • :wq! - 强制保存文件，退出vim。
  • :q - 不保存文件，退出vim。
  • :q! - 不保存文件，强制退出vim。
  • :e! - 放弃所有修改，从上次保存文件开始再编辑

使用gedit编辑器

  • gedit 文件名(创建新文件后会自动进入编辑器界面，输入内容，按<ctrl+s>保存文件内容，最后点击界面右上角的关闭按钮退出。)

退出命令行提示符

  • EXIT

Hexo 安装

Posted on 2022-05-30 Edited on 2022-06-03

安装Hexo

Hexo

  • npm install hexo-cli -g 安装Hexo

  • hexo -v 检查安装 如果输出的是一堆版本号那就安装成功了

  • 然后输入hexo init blog初始化文件夹，接着输入cd blog将目录切换到blog文件夹里,然后输入npm install就会在文件夹里生成一些需要的文件。

  • 这样就配置好了，可以输入hexo g生成静态网页，然后输入hexo s打开本地服务器来查看网页，
